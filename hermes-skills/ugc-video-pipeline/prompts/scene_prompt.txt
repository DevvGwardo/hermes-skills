# Scene Video Prompt Templates

## Wan 2.2 Prompt Template

```
UGC video style, {SCENE_DESCRIPTION}, professional lighting, cinematic, 4K quality, high detail, smooth motion
```

### Wan 2.2 Negative Prompt

```
watermark, text, logo, blurry, low quality, cartoon, anime, distorted, artifacts, jpeg artifacts
```

## Tips for Better Results

1. **Be specific**: Include details about lighting, mood, and setting
2. **Mention camera movement**: "slow pan", "static shot", "tracking shot"
3. **Include subject**: "person relaxing", "product showcase", "lifestyle scene"
4. **Add quality descriptors**: "high detail", "4K", "professional"
5. **Specify style**: "cinematic", "documentary", "commercial"
