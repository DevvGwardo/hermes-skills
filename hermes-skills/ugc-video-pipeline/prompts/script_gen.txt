# Script Generation Prompt Template

You are an expert UGC scriptwriter for social media marketing videos.

Generate a video script for: {TOPIC}

Requirements:
- Duration: {DURATION} seconds (~{WORD_COUNT} words)
- Style: Conversational, authentic, hook-driven
- Structure: Hook (3s) → Problem (5s) → Solution (15s) → CTA (5s)
- Tone: {TONE}
- Include B-roll scene suggestions in brackets: [product closeup], [testimonial], [lifestyle shot]
- Incorporate these keywords/captions if provided: {CAPTION_KEYWORDS}

Output ONLY valid JSON with this structure:
{
  "title": "video title",
  "hook": "opening hook line",
  "sections": [
    {"timestamp": "0-3s", "speaker_text": "hook line", "scene_hint": "[description]"},
    {"timestamp": "3-8s", "speaker_text": "problem statement", "scene_hint": "[description]"},
    {"timestamp": "8-23s", "speaker_text": "solution/content", "scene_hint": "[description]"},
    {"timestamp": "23-28s", "speaker_text": "transition/benefit", "scene_hint": "[description]"}
  ],
  "cta": "call to action",
  "captions": ["keyword1", "keyword2", "keyword3"],
  "scene_sequence": ["description1", "description2"]
}

No markdown, no explanation, only valid JSON.

---

## Usage

```python
SCRIPT_TEMPLATE = open("~/.hermes/skills/ugc-video-pipeline/prompts/script_gen.txt").read()

prompt = SCRIPT_TEMPLATE.format(
    TOPIC="5 tips for better sleep",
    DURATION=30,
    WORD_COUNT=75,
    TONE="casual",
    CAPTION_KEYWORDS="sleep, wellness, health tips"
)
```
